<div align="center"> <img src="https://oss.am-all.com.cn/asset/img/manual/controller/dasi/yubideck_00.jpg" width="400"/> </div>

!!! warning "注意事项"

    - **台式机** 必须使用主板后面的USB接口，前面板可能会出现供电不足，导致触摸失灵

    - 硬件安装顺序不能错！
        * 必须先固定两侧红外，再接HDMI线
        * 不然可能把HDMI线接口弄坏

### 接线

!!! info ""

    <div align="center"> <img src="https://oss.am-all.com.cn/asset/img/manual/controller/dasi/yubideck_04.jpg"/> </div>
        
    - 将 **HDMI线** 如图插入本体的插口内

    !!! warning "注意事项"

        - **此HDMI线不可接手台以外的其他设备！**
        - **此HDMI为手台专用通讯协议，并不能输出视频信号，也不兼容其他设备。**
        - **接到其他设备可能引起手台或其他设备的损坏！**

    - 将 **USB线** 如图插入本体的USB口，有两个口，插任意一个都是可以的
    - 空出来的USB接口作为备用，如果电脑供电严重不足，再用于额外供电（一般不会发生）。

    !!! note "USB线一定要插到底"

        - 感受到“咔”一下的手感才是正确的。
        - 如果插不到底，可能是外壳的孔没对准。
        - 将螺丝松开一点点，让USB插到底之后，再拧紧螺丝。

## 硬件功能说明

### 红外传感器 高度调整

!!! info ""

    <div align="center"> <img src="https://oss.am-all.com.cn/asset/img/manual/controller/dasi/yubideck_06.jpg"/> </div>

    - 红外感应器一共有8个灯，游戏中只会用到其中的6个。
    - 读卡器底部有个3档开关来选择，档1-最低6个；档2-中间6个；档3-最上6个。
    - 利用此功能，可以调整红外感应器的检测高度。

### 推荐值

!!! info ""

    - 坐着玩：1 或 2 档
    - 站着玩：2 或 3 档

### 功能键

!!! info ""

    - 触摸板正面的三个按键从左往右分别是A、B、C，具体功能会在说明书后面提到

## 一些参数

!!! warning "了解这些参数能让您更灵活的使用控制器，理解游戏特性"

### 触摸区块

!!! info ""

    <div align="center"> <img src="https://oss.am-all.com.cn/asset/img/manual/controller/dasi/yubideck_13.jpg"/> </div>

    - 触摸区域一共有32个触摸区块组成。

    - 每个轨道有上、下两个区块，如示意图所示排列。

### 触摸区块的差异

!!! info ""

    - 每个触摸区块之间会有制造差异，这是无法避免的。

    - 通过 **[校准模式](#_19)**，可将32个区块校准成相同的状态。

### 触摸灵敏度

!!! info ""

    - 在大四控制器上，触摸的灵敏度被划分为1～30档，数值越高，灵敏度越高。

    - 调整合适的灵敏度，可以避免以下问题

    !!! note "灵敏度太低"

        会导致手指按在4个区块交界处，没有反应。俗称中缝吃音。
        <div align="center"> <img src="https://oss.am-all.com.cn/asset/img/manual/controller/dasi/yubideck_14.jpg" width="200"/> </div>

        因为手指的触摸数值平均给了周围的区块，灵敏度太低会导致每个区块分得的数值达不到要求。

        **{==适当调高灵敏度==}** 就可解决此问题。

    ---

    !!! note "灵敏度太高"

        会导致粘键、手掌距离台子1cm以上就触发等问题。

        建议将灵敏度调低，且刚好不会出现 **{==中缝吃音==}** 的值为优。

    ---

    - 什么是 **合适的灵敏度**？

    !!! note "需同时符合以下条件"

        1. 用手指按4个区域交界处，有至少1个区块触发。

        2. 手掌放在离触摸区域1cm高度（一根手指的厚度）左右，没有区块触发。

### 硬件灵敏度

!!! info ""

    1. 这是硬件芯片内的一个参数，**{==一般情况下不需要改动它==}**。

    2. 当触摸灵敏度调整不出合适的值时，可以尝试修改此参数以增减硬件灵敏度。

    3. 使用此功能时，请务必重新进行一次 **{==触摸区块校准==}** ，因为硬件参数已被更改，原校准数据已不适用

    4. 如需调整请查看 **[调整硬件灵敏度](#_21)**

## 初次使用

!!! info "收到台子初次使用时，建议按以下顺序进行初次调整"

    ``` mermaid
    graph LR
      A[触摸区块校准] --> B[触摸灵敏度调整] --> C[开启游戏试玩] --> D[如有需要，再进行灵敏度调整];
    ```

!!! warning "注意事项"

    - 一般情况下，后续不需要再调整。

    - 每次更换环境（更换电脑、场地等），建议重新做一次区块校准。

## 控制器的特殊模式

### 设置模式

!!! info "进入此模式的方法"

    **{==同时长按功能键A+C==}** 1秒以上

!!! info "可设置内容"

    | 名称            | 描述                     |
    |-----------------|--------------------------|
    | `触摸区灯光颜色` | 可自定义按下、松开、间隔条颜色 |
    | `键盘模式键位数量` | 当 **USB输出模式** 为 **{==键盘信号==}** 时，可调整触摸区域划分为4、8、16、32键输出 |
    | `抬手传感器灯光颜色` | 可自定义抬手时、未抬手时灯光颜色 |
    | `键盘开启抬手信号` | 当 **USB输出模式** 为 **{==键盘信号==}** 时，可决定抬手是否输出键盘信号 |
    | `粘键抑制` | 如果因为温湿度变化太大、手套材质积累静电等原因，触摸区域发生粘键现象，可尝试开启此功能。 |
    | `USB输出模式` | 可切换USB模式为 **低延迟专用HID数据输出** 或 **键盘信号** 输出 |

    <div align="center"> <img src="https://oss.am-all.com.cn/asset/img/manual/controller/dasi/yubideck_09.jpg"/> </div>

    !!! warning "粘键抑制说明"
    
        - 粘键抑制开启后，不会影响判定和吃音。
    
        - 如果出现吃音问题，请优先调整控制器的 **[触摸灵敏度](#_13)**。
    
        - 增强型的粘键抑制，会将触摸数据的最大值限定在80附近。

!!! info "保存"

    - 长按功能键B键1秒以上以保存设定，并返回游玩状态

---

### 校准模式

!!! info "进入此模式的方法"

    **{==同时长按功能键A+B键==}** 1秒以上，进入成功后间隔条、侧板和LOGO灯的颜色会变为蓝色

    !!! warning "触摸板正面的三个按键从左往右分别是A、B、C"

!!! info "使用方法"

    **用尽量一致的按压力度，滑过全部32个键。**

    - 先滑完下层按键，滑过的按键会先亮蓝色灯

    <div align="center"> <img src="https://oss.am-all.com.cn/asset/img/manual/controller/dasi/yubideck_12.jpg"/> </div>

    <div align="center"> <img src="https://oss.am-all.com.cn/asset/img/manual/controller/dasi/yubideck_10.jpg"/> </div>

    - 全部亮绿灯1秒后会自动保存，并恢复正常游玩模式。

!!! info "其他说明"

    1. 蓝灯表示该轨道，识别到了其中1个区块有手指经过；绿灯表示2个。

    2. 因为此模式会自动放大灵敏度，可能会有部分轨道在滑一个区块的时候就会亮绿灯。请无视此现象，继续按照使用方法的说明正常滑完所有区域即可。

    3. 如果力度控制不好，想重新较准的话，按A键可清除记录值重新进行较准。

    4. 想要退出此模式且不保存，请同时长按A+B键1秒就会返回游玩状态。

    5. 如果力度太轻或手套太厚，以至于个别区块没有亮绿灯，可以长按C键强制保存较准结果。

---

### 触摸灵敏度调整模式

!!! info "进入此模式的方法"

    **{==同时长按B+C键==}** 1秒以上，进入成功后侧板和LOGO灯会熄灭

    !!! warning "触摸板正面的三个按键从左往右分别是A、B、C"

!!! info "使用方法"

    - 按B键，降低灵敏度等级；按C键，提高灵敏度等级

    - 根据间隔条数量代表灵敏度级别

    - 亮起 **{==红色==}** 间隔条的数量，代表范围是1~15

    - 亮起 **{==绿色==}** 间隔条的数量，代表范围是16~30

    - 等级越高，灵敏度越高

    !!! warning "可随时按触摸板，测试触摸效果。"

!!! info "保存"

    长按A键1秒以上以保存设定，并返回游玩状态。

!!! info "推荐值"

    - 常规感（需稍微用力打）

        灵敏度范围设定在10～15，即10～15个红灯

    ---

    - 轻松打

        灵敏度范围设定在16～20，即1～5个绿灯
---

### 调整硬件灵敏度

!!! warning "注意事项"

    建议先利用触摸灵敏度调整模式进行调整，除非实在调整不出来，再利用此功能

!!! info ""

    - 同时按住A+B键 (左侧2个)等待控制器进入校准模式

    <div align="center"> <img src="https://oss.am-all.com.cn/asset/img/manual/controller/dasi/yubideck_19.jpg" width="500"/> </div>

    - 进入后，再同时按住B+C键 (右侧2个)等待控制器进入内部参数设定模式

    <div align="center"> <img src="https://oss.am-all.com.cn/asset/img/manual/controller/dasi/yubideck_20.jpg" width="500"/> </div>

    - 按多次A键，调整蓝灯的个数。蓝灯数量为1~8个，数量越多，触摸越灵敏

    <div align="center"> <img src="https://oss.am-all.com.cn/asset/img/manual/controller/dasi/yubideck_21.jpg" width="500"/> </div>

    !!! note "推荐数量范围5～8"

    - 长按C键保存设定

    - **拔掉USB重新插入** ，再次按住A+B键 (左侧2个)进入校准模式，**重新校准即可使用**

## 硬件版本

!!! warning "由于经历2021年芯片灾情的原因，大四的芯片存在两种硬件版本"

### 硬件版本号查看

!!! info ""

    <div align="center"> <img src="https://oss.am-all.com.cn/asset/img/manual/controller/dasi/yubideck_15.jpg" width="500"/> </div>

    - 在触摸板背面的缝隙中可以看到硬件版本号

### 硬件版本号区分

!!! info ""

    - 硬件版本主要区分为 **无A版** 和 **有A版**

    | 版本        | 说明                                                    | 版本号范例  |
    |-----------|-------------------------------------------------------|--------|
    | **`无A版`** | 可自定义按下、松开、间隔条颜色                                       | V2.05  |
    | **`有A版`** | 当 **USB输出模式** 为 **{==键盘信号==}** 时，可调整触摸区域划分为4、8、16、32键输出 | V2A.05 |

    - 进行固件更新的时候，请注意区分以上两个版本。两个版本的固件不通用
